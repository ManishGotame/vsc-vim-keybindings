# Custom keybindings for vim and vscode

I use vim in vscode and here are some of my custom keybindings to improve my workflow.

## To apply the keybindings

1. Open Command Palette → Preferences: Open Settings (JSON)
2. Paste the config above inside your JSON.
3. Save the file and reload VSCode (or just the Vim extension).


## To apply the keyboard shortcuts keybindings
1. Open Command Palatte → Open Keyboard Shortcuts (JSON)
2. Past the config keyboard config.
3. Save the file.


Mainly used commands from here
| Command                     | Shortcut          |
|-----------------------------|-------------------|
| Toggle Vim                  | ⌥⌘V               |
| Increase editor view size   | ⌘⌥→               |
| Decrease editor view size   | ⌘⌥←               |
